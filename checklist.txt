- MOVIE API (8)
    GET /movies – danh sách phim
    GET /movies/:id – chi tiết phim
    POST /movies – thêm phim
    PUT /movies/:id – cập nhật phim
    DELETE /movies/:id – xóa phim
    GET /movies/:id/banners – danh sách banner của phim
    POST /movies/:id/banners – thêm banner cho phim
    DELETE /movies/:id/banners/:bannerId – xóa banner

- CINEMA API (13)
    GET /brands – danh sách thương hiệu rạp
    GET /cinemas – danh sách rạp
    GET /cinemas/:id – chi tiết rạp
    POST /cinemas – thêm rạp
    PUT /cinemas/:id – cập nhật rạp
    GET /cinemas/:id/screens – danh sách phòng chiếu
    POST /cinemas/:id/screens – thêm phòng chiếu
    GET /screens/:id/seats – danh sách ghế trong phòng
    POST /screens/:id/seats – thêm ghế
    GET /showtimes – danh sách suất chiếu
    GET /showtimes/:id – chi tiết suất chiếu
    POST /showtimes – thêm suất chiếu
    PUT /showtimes/:id – cập nhật suất chiếu

- USER API (4)
    POST /users/register – đăng ký
    POST /users/login – đăng nhập
    GET /users/:id – thông tin user
    PUT /users/:id – cập nhật user

- BOOKING API (3)
    POST /bookings – đặt vé
    GET /bookings/user/:userId – danh sách vé đã đặt của user
    DELETE /bookings/:id – hủy vé
